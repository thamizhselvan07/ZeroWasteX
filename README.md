## 🎯 Problem Statement

**Food waste is a global crisis.** Millions of tons of edible food are discarded daily, while millions face hunger.

Existing systems lack:

* ⚡ Real-time visibility of surplus food
* 🚚 Efficient coordination between donors and volunteers
* 🧑‍🤝‍🧑 Incentives for community participation
* 📊 Transparent impact tracking

---

## 💡 Solution — ZeroWasteX

**ZeroWasteX** transforms food redistribution into a **real-time, trackable, and scalable system**.

### 🔑 Key Capabilities:

1. **📍 Live Food Discovery**
   View available surplus food on an interactive map.

2. **🔄 Smart Pickup Workflow**
   Seamless lifecycle:
   `Available → Requested → Completed`

3. **🧠 Urgency-Based Prioritization**

   * 🔴 Urgent (< 1 hr)
   * 🟡 Soon (< 3 hrs)
   * 🟢 Safe

4. **🏆 Gamified Volunteer System**

   * Earn points for pickups
   * Leaderboard for top contributors

5. **📊 Impact Dashboard**

   * 🍱 Meals Saved
   * ♻️ Waste Reduced
   * 👥 People Fed

👉 **Result:**
More food rescued, less waste, stronger communities.

## ✨ Core Features

### 🗺️ Live Rescue Map

* Real-time food locations
* 🟢 Veg → Green markers
* 🔴 Non-Veg → Red markers

---

### 🔄 End-to-End Workflow

* Donors publish food
* Volunteers request pickup
* Track status until completion

---

### 📦 Pickup Management

* Dedicated **Pickup Page**
* **My Requests tracking**
* Real-time updates across all pages

---

### 🏆 Gamification

* +10 points per pickup
* +50 bonus for urgent rescue
* 🥇 Leaderboard system

---

### 📊 Impact Visualization

* Real-time statistics
* Global activity tracking
* Heatmap of food distribution

## 🚀 Local Development Setup

### 🔧 Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

(Optional: seed data)

```bash
python scripts/seed_sample_foods.py
```

---

### 🎨 Frontend

```bash
cd frontend
npm install
npm run dev
```
